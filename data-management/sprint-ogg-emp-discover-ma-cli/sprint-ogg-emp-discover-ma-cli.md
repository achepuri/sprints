# How do I discover GoldenGate MA instances using the EM CLI verb?
Duration: 1 minute

### Prerequisites
This sprint assumes you have:
  * Downloaded and Deployed the EM CLI Client

See **Learn More**.

## Discover GoldenGate Microservices (MA) instance

In the EM CLI client, run the following command:
```
<copy>
emcli discover_ggma
[-configFile="configFile"]
[-debug]
[-check]
</copy>
```
